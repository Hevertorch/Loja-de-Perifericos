##  Sobre o Projeto

Projeto web desenvolvido em **PHP**, seguindo o **padrão arquitetural MVC (Model–View–Controller)**, simulando uma **loja de periféricos**.  
O objetivo é aplicar conceitos de **backend, frontend, banco de dados e organização de código**, mantendo uma estrutura clara, reutilizável e escalável.

---

##  Funcionalidades

- Cadastro e listagem de periféricos
- Páginas diferentes para cada tipo de usuário (cliente e administrador)
- Botão de lista de desejos conectado individualmente a cada perfil de usuário (a lista de desejos de um usuário não interfere na de outro) 
- Separação clara entre:
  - **Model** — regras de negócio e acesso ao banco  
  - **View** — interface com o usuário  
  - **Controller** — controle do fluxo da aplicação  
- Interface responsiva com **Bootstrap**  
- Interações dinâmicas com **JavaScript**  
- Persistência de dados com **MySQL**

## Tecnologias Utilizadas
